Configuration pour l'utilisation de ResPinGuy

Minimum Ram : 4 Go | Recomander : 8 Go
Minimun CPU : 4 Core 2.50 GHz | Recomander CPU : 8 Core 3.50 GHz
Minimum Storage : 50 Go Libre HDD | Recomander : 100 Go Libre SSD

_ _ _


DSINRD pour Ubuntu :
http://respinguy.tk/ftp/file-respinguy/initrd/officiel/Ubuntu/make-initrd.zip

Le noyaux de Ubuntu qui est installer par defaut n'est pas compatible avec ResPinGuy.
Vous pouvez envisager d'installer un autre noyaux ou de le recompiler.
Nous vous recommandons le noyaux Liquorix (https://liquorix.net)
